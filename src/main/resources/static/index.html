Google

功能支持
Campaign: 按天、按Campaign&天统计
文件格式要求
csv
列名要求
日期列:"Day"或者"天"
Campaign列:"Campaign"
不可用事件列名："Unavailable_SKAdNetwork Conversions" 或 "不可用_SKAdNetwork 转化次数"
可用事件列名："数字_SKAdNetwork Conversions" 或 "数字_SKAdNetwork 转化次数"，其中数字为事件号

Facebook

功能支持
Campaign：按天、按Campaign&天统计
素材：按素材&天、素材&周、素材&月统计
文件格式要求
xls，xlsx
列名要求
天
广告系列名称（按Campaign统计时必须）
图片、视频和幻灯片（按素材统计时必须）
花费金额 (USD)
展示次数
点击量（全部）
应用安装
购物
购物转化价值
